# Skeleton UI React 프로젝트

React와 Vite를 사용한 Skeleton UI 데모 프로젝트입니다.

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 주요 기능

- 카드 스켈레톤
- 리스트 아이템 스켈레톤
- 이미지 카드 스켈레톤
- 로딩 애니메이션
- 실제 콘텐츠 전환

## 사용 기술

- React 18
- Vite
- CSS3 Animations
